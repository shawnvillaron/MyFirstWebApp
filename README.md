# Hello World App

This is a simple Hello World application that demonstrates basic functionality.

## Features
- Displays "Hello, World!" message
- User-friendly interface

## Installation
1. Clone the repository: `git clone https://github.com/shawnvillaron/MyFirstWebApp`
2. Navigate to the project directory.
3. Open `index.html` in your web browser.

## Usage
Simply open the application in a web browser to see the output!
